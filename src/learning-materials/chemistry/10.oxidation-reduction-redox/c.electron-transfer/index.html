Aufbau Principle

Electrons occupy the lowest energy orbitals first. This "building-up" principle explains the sequential filling order: 1s → 2s → 2p → 3s → 3p → 4s → 3d → 4p...

Hund's Rule

Electrons fill degenerate orbitals (orbitals with the same energy) singly first, with parallel spins, before pairing up. This minimizes electron-electron repulsion.

Pauli Exclusion

No two electrons in an atom can have the same set of four quantum numbers. This means each orbital can hold at most two electrons with opposite spins.
