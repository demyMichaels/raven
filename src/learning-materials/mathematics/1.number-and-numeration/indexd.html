üìö Understanding Binary

How Binary Works:

Binary uses only 0s and 1s. Each LED represents a power of 2:

  • LED 0 (rightmost): 2‚Å∞ = 1
  • LED 1: 2¬π = 2
  • LED 2: 2¬≤ = 4
  • LED 3 (leftmost): 2¬≥ = 8

Converting Binary to Decimal:

Add up the values where LEDs are ON (1).

Example: 1101 = 8 + 4 + 0 + 1 = 13
