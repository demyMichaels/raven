Best Practices & Tips

✓ Do's

  • • Use cubic-bezier easing for smooth, natural motion
  • • Keep animations between 200-400ms for optimal feel
  • • Add backdrop overlay to focus attention on menu
  • • Use transform for better performance than position changes
  • • Include close button and overlay click to dismiss
  • • Add stagger animations for menu items (50-100ms delay)

✗ Don'ts

  • • Avoid animations longer than 500ms - they feel sluggish
  • • Don't use linear easing - it feels mechanical
  • • Avoid too many simultaneous effects - keep it simple
  • • Don't forget touch targets - min 44x44px for mobile
  • • Never disable body scroll without good reason
  • • Avoid 3D effects on low-end devices - test performance

Performance Tips

Use GPU Acceleration

Add will-change: transform for smoother animations

Optimize Repaints

Prefer transform and opacity over other properties

Test on Real Devices

Always test animations on actual mobile hardware

Accessibility Considerations

Keyboard Navigation

  • → Ensure menu can be opened/closed with Enter or Space
  • → Add Escape key handler to close menu
  • → Trap focus within menu when open

ARIA Attributes

  • → Use aria-expanded on menu button
  • → Add role="navigation" to menu
  • → Include aria-label for screen readers

Reduced Motion

@media (prefers-reduced-motion: reduce) {
    .mobile-menu {
        transition: none;
    }
}

Color Contrast

  • → Ensure 4.5:1 contrast ratio for text
  • → Test both light and dark mode

Complete JavaScript Implementation

Here's a production-ready JavaScript implementation with all the best practices:

// Mobile Menu Handler
class MobileMenu {
    constructor(options) {
        this.menu = document.querySelector(options.menuSelector);
        this.trigger = document.querySelector(options.triggerSelector);
        this.overlay = document.querySelector(options.overlaySelector);
        this.closeBtn = document.querySelector(options.closeSelector);
        this.isOpen = false;

        this.init();
    }

    init() {
        // Click handlers
        this.trigger?.addEventListener('click', () => this.toggle());
        this.closeBtn?.addEventListener('click', () => this.close());
        this.overlay?.addEventListener('click', () => this.close());

        // Keyboard handlers
        document.addEventListener('keydown', (e) => {
            if (e.key === 'Escape' && this.isOpen) this.close();
        });

        // Prevent body scroll when menu is open
        this.menu?.addEventListener('transitionend', () => {
            if (this.isOpen) {
                this.trapFocus();
            }
        });
    }

    toggle() {
        this.isOpen ? this.close() : this.open();
    }

    open() {
        this.menu?.classList.add('open');
        this.overlay?.classList.add('open');
        this.trigger?.setAttribute('aria-expanded', 'true');
        document.body.style.overflow = 'hidden';
        this.isOpen = true;

        // Stagger menu items
        this.animateMenuItems();
    }

    close() {
        this.menu?.classList.remove('open');
        this.overlay?.classList.remove('open');
        this.trigger?.setAttribute('aria-expanded', 'false');
        document.body.style.overflow = '';
        this.isOpen = false;
    }

    animateMenuItems() {
        const items = this.menu?.querySelectorAll('li');
        items?.forEach((item, index) => {
            setTimeout(() => {
                item.style.opacity = '1';
                item.style.transform = 'translateX(0)';
            }, index * 50);
        });
    }

    trapFocus() {
        const focusableElements = this.menu?.querySelectorAll(
            'a, button, input, textarea, select, [tabindex]:not([tabindex="-1"])'
        );
        const firstElement = focusableElements?.[0];
        const lastElement = focusableElements?.[focusableElements.length - 1];

        firstElement?.focus();

        this.menu?.addEventListener('keydown', (e) => {
            if (e.key === 'Tab') {
                if (e.shiftKey && document.activeElement === firstElement) {
                    e.preventDefault();
                    lastElement?.focus();
                } else if (!e.shiftKey && document.activeElement === lastElement) {
                    e.preventDefault();
                    firstElement?.focus();
                }
            }
        });
    }
}

// Initialize menu
const menu = new MobileMenu({
    menuSelector: '.mobile-menu',
    triggerSelector: '.menu-trigger',
    overlaySelector: '.menu-overlay',
    closeSelector: '.menu-close'
});
