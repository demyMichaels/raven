Syntax:

box-shadow: offset-x offset-y blur-radius spread-radius color;
offset-x:

Horizontal distance (+ = right, - = left)

offset-y:

Vertical distance (+ = down, - = up)

blur-radius:

How blurry the shadow is (larger = softer)

spread-radius:

Size expansion (optional, + = larger, - = smaller)

ðŸŽ¨ Material Design Elevation System

Material Design uses elevation to create depth and hierarchy. Each elevation level uses multiple shadow layers for realistic depth perception.

ðŸ’¡ Pro Tip: Material shadows combine two layers - an umbra (sharp shadow) and a penumbra (soft shadow) - to mimic real-world lighting.
