What Are Progress Rings?

Progress rings (also called circular progress indicators) are elegant UI components that visualize completion, loading states, or statistics in a circular format. They're created using SVG circles with clever manipulation of the stroke-dasharray and stroke-dashoffset properties.

The Math Behind Progress Rings

Step 1: Circle Circumference

First, calculate the circle's circumference using the radius:

// Formula: 2 × π × radius const radius = 40; const circumference = 2 * Math.PI * radius; // Result: 251.2

Step 2: Set Dash Array

Set stroke-dasharray to the circumference:

.progress-ring-circle { stroke-dasharray: 251.2; /* Creates a dash equal to circumference */ }

This creates a single dash that exactly wraps around the circle, making it appear as a complete circle.

Step 3: Calculate Offset

For a given percentage, calculate the offset:

const percent = 75; // 75% complete const offset = circumference - (percent / 100) * circumference; // offset = 251.2 - (0.75 × 251.2) = 62.8

The offset "pushes" the dash around the circle, revealing the progress.

Step 4: Rotate -90°

Rotate the ring so it starts at the top:

.progress-ring { transform: rotate(-90deg); /* SVG circles start at 3 o'clock */ /* -90deg rotation moves start to 12 o'clock */ }

Design Best Practices

✓

Use Round Line Caps

stroke-linecap="round" creates smoother, more polished endpoints

✓

Smooth Transitions

Use easing functions like cubic-bezier(0.4, 0, 0.2, 1) for natural motion

✓

Adequate Contrast

Ensure sufficient contrast between progress and background, especially in dark mode

✓

Meaningful Colors

Use green for completion, orange for warnings, red for errors or critical states

✗

Avoid Thin Strokes on Mobile

Stroke widths below 4px can be hard to see on small screens

✗

Don't Forget Accessibility

Always include text labels or ARIA attributes for screen readers

✗

Avoid Overly Complex Animations

Respect prefers-reduced-motion for users who need it

✗

Don't Use Only Color

Combine color with text, icons, or patterns for colorblind users

Reusable JavaScript Function

/** * Updates a progress ring to a specific percentage * @param {SVGCircleElement} circle - The progress circle element * @param {number} percent - Progress percentage (0-100) */ function setProgress(circle, percent) { const radius = circle.r.baseVal.value; const circumference = 2 * Math.PI * radius; const offset = circumference - (percent / 100) * circumference; circle.style.strokeDasharray = `${circumference}`; circle.style.strokeDashoffset = `${offset}`; } // Usage example: const progressCircle = document.querySelector('.progress-ring-circle'); setProgress(progressCircle, 75); // Set to 75% // Animate from 0 to 100: let progress = 0; const interval = setInterval(() => { progress += 1; setProgress(progressCircle, progress); if (progress >= 100) clearInterval(interval); }, 20);
