What is Material Design Typography?

Material Design's typography scale provides a structured system of 13 type styles that work together to create hierarchy and communicate your brand effectively across all screen sizes.

The system is mobile-first, meaning base sizes are optimized for mobile devices, with larger variants applied on desktop screens (768px+). This ensures readability and visual harmony across all devices.

Key Principles

Mobile-First Approach

Base sizes are optimized for mobile devices (3.5rem for headline-1), then scale up for desktop (6rem at 768px+). This ensures readability on smaller screens first.

Visual Hierarchy

Each style has a specific purpose in the hierarchy. Headlines create structure, body text delivers content, and utility styles provide context and actions.

Consistent Spacing

Letter spacing and line height are carefully calibrated for each style to ensure optical balance and readability across all sizes.

Purposeful Weights

Font weights (300-500) are chosen strategically: lighter weights for large headlines, medium weights for emphasis and buttons.
