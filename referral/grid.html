1. Grid Basics

CSS Grid is a two-dimensional layout system that allows you to create complex layouts with rows and columns. Let's start with the fundamentals.

Key Concept:

Grid creates a parent-child relationship where the parent is the grid container and children are grid items.

Simple Grid Example

2. Tailwind Grid Classes

Display Classes

  • grid - Creates a grid container
  • inline-grid - Creates an inline grid container

Column Classes

  • grid-cols-{n} - n columns (1-12)
  • grid-cols-none - No explicit columns

Row Classes

  • grid-rows-{n} - n rows (1-6)
  • grid-rows-none - No explicit rows

Gap Classes

  • gap-{n} - Gap between items
  • gap-x-{n} - Horizontal gap
  • gap-y-{n} - Vertical gap

6. Grid Template Areas

Template areas allow you to name grid areas and place items by name. This requires custom CSS.

Header
Sidebar
Main Content
Footer
<style> .template-area-grid { display: grid; grid-template-areas: 'header header header' 'sidebar main main' 'footer footer footer'; gap: 1rem; } .header { grid-area: header; } .sidebar { grid-area: sidebar; } .main { grid-area: main; } .footer { grid-area: footer; } </style>

7. Auto-fit & Auto-fill

Note:

Tailwind doesn't have built-in classes for auto-fit/auto-fill, but you can use arbitrary values or custom CSS.

Auto-fit (Items stretch to fill space)

1
2
3
4
<div style="display: grid; grid-template-columns: repeat(auto-fit, minmax(150px, 1fr)); gap: 1rem;"> <!-- Items will stretch to fill available space --> </div> <!-- Or with Tailwind arbitrary values: --> <div class="grid gap-4" style="grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));">

Auto-fill (Creates empty tracks)

1
2
3
<div style="display: grid; grid-template-columns: repeat(auto-fill, minmax(150px, 1fr)); gap: 1rem;"> <!-- Creates as many tracks as will fit --> </div>
